# Professional Portfolio Website

A clean, responsive portfolio website template built with HTML, CSS, and JavaScript.

## Files

- `index.html` — homepage layout and sections (hero, about, projects, process, testimonials, skills, contact)
- `styles.css` — responsive styling with dark theme and mobile optimization
- `script.js` — mobile navigation toggle and contact form handling
- `favicon.svg` — branded icon

## Quick Start

### Local Preview

```bash
# Python 3
python -m http.server 8000

# Then visit http://localhost:8000
```

### Customization

1. **Update your name and contact info:**
   - Replace `KISHORE KANNA S` with your name in header and footer
   - Update `hello@kishore.dev` with your email in contact section
   - Update social links (LinkedIn, GitHub) with your profiles

2. **Replace placeholder content:**
   - Hero headline and copy
   - About section and experience
   - Project titles and descriptions
   - Skills and process steps
   - Testimonials and client names

3. **Update metadata for SEO:**
   - `<meta name="description">` — your tagline
   - `og:title` and `og:description` — social sharing
   - `og:url` — your domain URL

## Deployment

### GitHub Pages (Free)

1. Create a repository named `yourname.github.io`
2. Push this portfolio to the repo
3. Visit `https://yourname.github.io`

### Netlify (Free)

1. Push to GitHub
2. Connect to Netlify and auto-deploy
3. Add custom domain if you have one

### Vercel (Free)

1. Import from GitHub
2. Deploy with one click
3. Custom domain support
